## Instruções para uso

### 1º Passo
#### Clone o repositório
```
git clone https://github.com/JoaoVictor087/DevOps-CP04.git
```

### 2º Passo
#### Entre no diretório
```
cd /DevOps-CP04
```

### 3º Passo
#### Execute o container
#### obs: caso esteja no Windows, abra o Docker Desktop
```
docker compose --file compose.yaml up --build
```

### 4º Passo
#### Teste a aplicação na web
```
localhost:8081/hello
```
